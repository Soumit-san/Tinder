# Sentix Tinder — Frontend

This is the React 19 + Vite frontend for Sentix Tinder, displaying sentiment analytics, trends, word clouds, and fake/spam detection data.

Please see the [Root README](../README.md) for full project architecture, features, and deployment instructions.

## Development

```bash
npm install
npm run dev
```

The frontend runs on `http://localhost:5173` by default and connects to the FastAPI backend at `http://localhost:8000/api`.
